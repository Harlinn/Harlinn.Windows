# Power and root finding functions

[Optimized basic math functions](Readme.md)

- [Cbrt](Cbrt.md) Calculates the cube root of the argument.
- [EvaluatePolynomial](./EvaluatePolynomial.md): Horner method with compensated evaluation via `FMA`.
- [FastPow](./FastPow.md): Computes v raised to the integer power n using exponentiation by squaring.
- [FastExp](./FastExp.md): an efficient approximation for `e^x`.
- [Hypot](Hypot.md) Calculates the hypotenuse of the arguments.
- [Pow](Pow.md) Calculates x raised to the power of y.
- [ReciprocalSqrt](ReciprocalSqrt.md) Calculates the reciprocal of the square root of the argument.
- [Sqrt](Sqrt.md) Calculates the square root of the argument.
- [Sqr](Sqr.md) Calculates the square of the argument.
- [Quadratic](./Quadratic.md): robust quadratic solver using compensated/discriminant checks.
- [NewtonBisection](./NewtonBisection.md): hybrid root-finding with Newton and bisection.
