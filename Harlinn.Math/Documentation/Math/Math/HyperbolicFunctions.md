# Hyperbolic functions

[Optimized basic math functions](Readme.md)

- [ACosH](ACosH.md) Returns the inverse hyperbolic cosine of the argument.
- [ASinH](ASinH.md) Returns the inverse hyperbolic sine of the argument.
- [ATanH](ATanH.md) Returns the inverse hyperbolic tangent of the argument.
- [CosH](CosH.md) Returns the hyperbolic cosine of the argument.
- [SinH](SinH.md) Returns the hyperbolic sine of the argument.
- [TanH](TanH.md) Returns the hyperbolic tangent of the argument.
