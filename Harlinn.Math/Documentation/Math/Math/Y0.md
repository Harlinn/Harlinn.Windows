# Y0

## Summary

`Y0` computes the Bessel function of the second kind of order zero (also called Neumann function or Weber function) for a floating-point argument. It delegates to the library's `OpenLibM` helpers and is `constexpr`-friendly when those helpers are available in constant-evaluation contexts.

(Defined in `Harlinn::Math`, header: `Harlinn.Math\include\Harlinn\Math\Math.h`.)

## Declaration

```cpp
template<FloatingPointType T>
constexpr inline T Y0( T x ) noexcept;
```

## Behavior

- Returns the value of the Bessel function of the second kind of order 0, Y0(x), for real argument `x > 0`.
- For `float` the implementation delegates to `Math::Internal::OpenLibM::y0f`; for `double` it delegates to `Math::Internal::OpenLibM::y0`.
- The function is `noexcept` and follows the semantics of the underlying OpenLibM implementation for special values (NaN, infinities, and domain errors).
- Y0(x) is singular at x = 0 (it tends to -?) and oscillatory for large |x| with amplitude decaying roughly as sqrt(2/(?|x|)).

## Notes

- Use `Y0` when solving problems involving cylindrical wave equations where the second-kind Bessel function appears (e.g., in boundary-value problems with singular behaviour at the origin).
- For integer order relations and other orders, use `Y1` or `YN`.

## Examples

Runtime example:

```cpp
#include <iostream>
#include <limits>
#include "Harlinn/Math/Math.h" // adjust include path as needed

int main()
{
    using Harlinn::Math::Y0;

    double x1 = 0.5;
    double x2 = 1.0;
    double x3 = 5.0;

    std::cout << "Y0(0.5) = " << Y0( x1 ) << '\n';
    std::cout << "Y0(1.0) = " << Y0( x2 ) << '\n';
    std::cout << "Y0(5.0) = " << Y0( x3 ) << '\n';

    double nan = std::numeric_limits<double>::quiet_NaN();
    std::cout << "Y0(NaN) is NaN: " << std::isnan( Y0( nan ) ) << '\n';

    return 0;
}
```

Compile-time example (constexpr-friendly):

```cpp
#include "Harlinn/Math/Math.h"

constexpr double y0_at_1 = Harlinn::Math::Y0( 1.0 );
// Y0(1) ~ 0.08825696421567697
static_assert( y0_at_1 > 0.0882569642156769 && y0_at_1 < 0.0882569642156771 );

// Note: Y0 is singular at 0 and not defined as a finite value there.
```
