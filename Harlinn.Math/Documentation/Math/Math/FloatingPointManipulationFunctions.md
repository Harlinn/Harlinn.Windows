# Floating point manipulation functions

[Optimized basic math functions](Readme.md)

- [FRExp](FRExp.md) Retrieves the mantissa and exponent of a floating-point number.
- [ModF](ModF.md) Splits the floating-point argument into fractional and integer parts.
- [NextAfter](NextAfter.md) Calculates next representable double-precision floating-point number.
- [NextDown](NextDown.md) Return the largest floating point number y of the same type as x such that $y < x$.  
- [NextUp](NextUp.md) Return the smallest floating point number y of the same type as x such that $y > x$. 
- [SignBit](SignBit.md) Determines if the floating point argument is negative.
- [signum](signum.md) Returns the sign of the argument.
