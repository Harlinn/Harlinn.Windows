# Basic operations

[Optimized basic math functions](Readme.md)

- [Abs](Abs.md) Calculates the absolute value of a floating point or integral value $\lvert x\rvert$.
- [FastAbs](FastAbs.md) Calculates the absolute value of a floating point or integral value $\lvert x\rvert$.
- [FMod](FMod.md) Calculates the remainder of a floating point division operation.
- [Remainder](Remainder.md) Calculates the signed remainder of a floating point division operation.
- [RemQuo](RemQuo.md) Calculates the signed remainder including the three last bits of the division operation.
- [FMA](FMA.md) Calculates the result of the fused multiply-add operation.
