# Special Functions

[Optimized basic math functions](Readme.md)

- [J0](./J0.md) Returns the Bessel function of the first kind of order 0.
- [J1](./J1.md) Returns the Bessel function of the first kind of order 1.
- [JN](./JN.md) Returns the Bessel function of the first kind of order N.
- [Y0](./Y0.md) Returns the Bessel function of the second kind of order 0.
- [Y1](./Y1.md) Returns the Bessel function of the second kind of order 1.
- [YN](./YN.md) Returns the Bessel function of the second kind of order N.
