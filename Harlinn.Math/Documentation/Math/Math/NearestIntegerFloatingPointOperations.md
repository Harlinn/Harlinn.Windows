# Nearest integer floating point operations

[Optimized basic math functions](Readme.md)

- [Ceil](Ceil.md) Returns the largest integral value greater than or equal to the argument.
- [Floor](Floor.md) Returns the largest integral value less than or equal to the argument.
- [Round](Round.md) Rounds the floating-point argument to the nearest integral value.
- [Trunc](Trunc.md) Returns the nearest integral value not greater in magnitude than the argument.
