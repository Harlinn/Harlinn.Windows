# Classification and comparison

[Optimized basic math functions](Readme.md)

- [IsNaN](IsNaN.md) Checks if the argument is NaN.
- [IsNormal](IsNormal.md) Checks if the argument is a normal value.
- [IsZero](IsZero.md) Checks if the argument is zero.
