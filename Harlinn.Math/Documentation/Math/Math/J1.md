# J1

## Summary

`J1` computes the Bessel function of the first kind of order one for a floating-point argument. It is constexpr-friendly via the library's `OpenLibM` helpers and follows the standard mathematical semantics for the Bessel J1 function.

(Defined in `Harlinn::Math`, header: `Harlinn.Math\include\Harlinn\Math\Math.h`.)

## Declaration

```cpp
template<FloatingPointType T>
constexpr inline T J1( T x ) noexcept;
```

## Behavior

- Returns the value of the Bessel function of the first kind of order 1, J1(x), for the argument `x` (real-valued).
- For `float` the implementation delegates to `Math::Internal::OpenLibM::j1f`; for `double` it delegates to `Math::Internal::OpenLibM::j1`.
- The function is `noexcept` and can be used in constant-evaluation contexts when the underlying helpers are constexpr-capable.
- For large |x| the function oscillates and decays in amplitude roughly as sqrt(2/(?|x|)) with phase shift; near x = 0, J1(x) ~ x/2 and J1(0) = 0.
- Special values follow underlying library semantics: NaN inputs propagate NaN; infinities are handled per the OpenLibM implementation.

## Notes

- `J1` appears in problems with cylindrical symmetry and in solutions to differential equations related to wave propagation.
- Use `JN(n, x)` for Bessel functions of other integer orders.

## Examples

Runtime example:

```cpp
#include <iostream>
#include "Harlinn/Math/Math.h" // adjust include path as needed

int main()
{
    using Harlinn::Math::J1;

    double x1 = 0.0;
    double x2 = 1.0;
    double x3 = 5.0;

    std::cout << "J1(0) = " << J1( x1 ) << '\n';   // 0
    std::cout << "J1(1) = " << J1( x2 ) << '\n';   // ~0.440050586
    std::cout << "J1(5) = " << J1( x3 ) << '\n';   // ~-0.327579138

    return 0;
}
```

Compile-time example (constexpr-friendly):

```cpp
#include "Harlinn/Math/Math.h"

constexpr double j1_0 = Harlinn::Math::J1( 0.0 );
static_assert( j1_0 == 0.0 );

constexpr double j1_1 = Harlinn::Math::J1( 1.0 );
// J1(1) ~ 0.4400505857449335
static_assert( j1_1 > 0.4400505857 && j1_1 < 0.4400505858 );
```
