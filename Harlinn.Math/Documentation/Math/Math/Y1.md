# Y1

## Summary

`Y1` computes the Bessel function of the second kind of order one (also called the Neumann function) for a floating-point argument. It delegates to the library's `OpenLibM` helpers and supports constant-evaluation when those helpers are available.

(Defined in `Harlinn::Math`, header: `Harlinn.Math\include\Harlinn\Math\Math.h`.)

## Declaration

```cpp
template<FloatingPointType T>
constexpr inline T Y1( T x ) noexcept;
```

## Behavior

- Returns the value of the Bessel function of the second kind of order 1, Y1(x), for real argument `x > 0`.
- For `float` the implementation delegates to `Math::Internal::OpenLibM::y1f`; for `double` it delegates to `Math::Internal::OpenLibM::y1`.
- The function is `noexcept` and follows the semantics of the underlying OpenLibM implementation for special values (NaN, infinities, and domain errors).
- Y1(x) has a singularity at x = 0 (tends to -?) and oscillates for large |x| with amplitude decaying roughly as sqrt(2/(?|x|)).

## Notes

- `Y1` often appears together with `J1` when forming general solutions to cylindrical wave problems; `Y1` provides the independent solution that is singular at the origin.
- For integer order relations and other orders, use `Y0` or `YN`.

## Examples

Runtime example:

```cpp
#include <iostream>
#include <limits>
#include "Harlinn/Math/Math.h" // adjust include path as needed

int main()
{
    using Harlinn::Math::Y1;

    double x1 = 0.5;
    double x2 = 1.0;
    double x3 = 5.0;

    std::cout << "Y1(0.5) = " << Y1( x1 ) << '\n';
    std::cout << "Y1(1.0) = " << Y1( x2 ) << '\n';
    std::cout << "Y1(5.0) = " << Y1( x3 ) << '\n';

    double nan = std::numeric_limits<double>::quiet_NaN();
    std::cout << "Y1(NaN) is NaN: " << std::isnan( Y1( nan ) ) << '\n';

    return 0;
}
```

Compile-time example (constexpr-friendly):

```cpp
#include "Harlinn/Math/Math.h"

// Note: Y1 is singular at 0; use positive arguments for constexpr evaluation
constexpr double y1_at_1 = Harlinn::Math::Y1( 1.0 );
// Y1(1) ~ -0.7812128213002887
static_assert( y1_at_1 > -0.78121282130029 && y1_at_1 < -0.78121282130028 );
```
