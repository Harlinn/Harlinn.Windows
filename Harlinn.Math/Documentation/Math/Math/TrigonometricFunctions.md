# Trigonometric functions

[Optimized basic math functions](Readme.md)

- [ACos](ACos.md) Returns the arccosine of the argument.
- [ASin](ASin.md) Returns the arcsine of the argument.
- [ATan](ATan.md) Returns the arctangent of the argument.
- [ATan2](ATan2.md) Returns the arctangent of the arguments (x/y).
- [Cos](Cos.md) Returns the cosine of the argument.
- [Deg2Rad](Deg2Rad.md) Converts the argument from degrees to radians.
- [Rad2Deg](Rad2Deg.md) Converts the argument from radians to degrees.
- [Sin](Sin.md) Returns the sine of the argument.
- [Tan](Tan.md) Returns the tangent of the argument.
