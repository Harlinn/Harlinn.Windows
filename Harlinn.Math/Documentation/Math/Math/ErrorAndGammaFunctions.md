# Error and gamma functions

[Optimized basic math functions](Readme.md)

- [Erf](Erf.md) Calculates the error function.
- [ErfC](ErfC.md) Calculates the complementary error function.
- [Gamma](Gamma.md) Estimates the maximum relative error after `n` floating-point operations.
- [TGamma](TGamma.md) Calculates the gamma function.
- [LGamma](LGamma.md) Calculates the natural logarithm of the gamma function.
