# DmXeSessionsReader — reference

Overview

`DmXeSessionsReader` wraps `sys.dm_xe_sessions` and returns information about active Extended Events sessions, including buffer usage and generation statistics.

Reader SQL

```
SELECT
  dxs.[Address],
  dxs.[Name],
  dxs.[pending_buffers],
  dxs.[total_regular_buffers],
  dxs.[regular_buffer_size],
  dxs.[total_large_buffers],
  dxs.[large_buffer_size],
  dxs.[total_buffer_size],
  dxs.[buffer_policy_flags],
  dxs.[buffer_policy_desc],
  dxs.[Flags],
  dxs.[flag_desc],
  dxs.[dropped_event_count],
  dxs.[dropped_buffer_count],
  dxs.[blocked_event_fire_time],
  dxs.[create_time],
  dxs.[largest_event_dropped_size],
  dxs.[session_source],
  dxs.[buffer_processed_count],
  dxs.[buffer_full_count],
  dxs.[total_bytes_generated]
FROM
  [sys].[dm_xe_sessions] dxs
```

Columns and interpretation

- `Address` — binary address for the session instance.
- `Name` — session name.
- `pending_buffers`, `total_regular_buffers`, `regular_buffer_size`, `total_large_buffers`, `large_buffer_size`, `total_buffer_size` — buffer allocation and sizing statistics.
- `buffer_policy_flags`, `buffer_policy_desc` — buffer policy flags and description.
- `Flags`, `flag_desc` — session flags and description.
- `dropped_event_count`, `dropped_buffer_count` — counts of dropped events and buffers due to resource limits.
- `blocked_event_fire_time` — time the session was blocked from firing events.
- `create_time` — session creation time.
- `largest_event_dropped_size` — size of the largest event dropped.
- `session_source` — source description for the session (e.g., user-defined or system).
- `buffer_processed_count`, `buffer_full_count` — counters of processed and full buffers.
- `total_bytes_generated` — total bytes of event data generated by the session.

How to interpret

- These metrics help monitor Extended Events session health; high dropped counts or buffer full counts indicate session throttling or misconfiguration.

Example

```csharp
using var cmd = conn.CreateCommand();
cmd.CommandText = DmXeSessionsReader.Sql;
using var rdr = cmd.ExecuteReader();
var r = new DmXeSessionsReader(rdr, ownsReader:false);
while (r.Read())
{
    Console.WriteLine($"Session:{r.Name} PendingBuffers:{r.PendingBuffers} DroppedEvents:{r.DroppedEventCount} TotalBytes:{r.TotalBytesGenerated}");
}
```
