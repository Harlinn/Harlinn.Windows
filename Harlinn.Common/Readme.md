Harlinn.Common is a header only library which only depends on a 
few standard C++ header files:

- `<bit>`
- `<cassert>`
- `<cmath>`
- `<iterator>`
- `<limits>`
- `<optional>`
- `<string_view>`
- `<type_traits>`

The library provides functionality used by the Harlinn.Math library and the 
Harlinn.Common.Core library.
