# Harlinn.Windows

This is the repository for Harlinn.Windows, a Visual Studio 2026 Solution with a wide range of useful C and C++ libraries for 64-bit Windows software development. 

The code was originally published on [CodeProject.com](https://www.codeproject.com/Articles/5319700/Harlinn-Windows-A-Visual-Studio-2022-Solution) 
in 2021, and has a history that goes back to 2012.

.Net and C# libraries and tools:
- [Harlinn.Common.Core.Net](https://github.com/Harlinn/Harlinn.Windows/tree/master/DotNet/Libraries/Harlinn.Common.Core.Net):
  - Classes for working with the [Microsoft.Data.SqlClient](https://learn.microsoft.com/en-us/sql/connect/ado-net/microsoft-ado-net-sql-server) library, 
    providing a strongly typed interface for accessing SQL Server metadata. Used to implement [Harlinn.MSSql.Tool](https://github.com/Harlinn/Harlinn.Windows/tree/master/DotNet/Tools/Harlinn.MSSql.Tool).
  - Classes used by the C# code generated by [dbxgen](https://github.com/Harlinn/Harlinn.Windows/tree/master/Tools/Databases/dbxgen) tool.
  - Classes used by the [Harlinn.Hydrology.Net](https://github.com/Harlinn/Harlinn.Windows/tree/master/DotNet/Libraries/Harlinn.Hydrology.Net)

- [Harlinn.MSSql.Tool](https://github.com/Harlinn/Harlinn.Windows/tree/master/DotNet/Tools/Harlinn.MSSql.Tool) a command line tool for generating C# code for working with Microsoft SQL Server databases using the 
  [Microsoft.Data.SqlClient](https://learn.microsoft.com/en-us/sql/connect/ado-net/microsoft-ado-net-sql-server) library.

- [Harlinn.Hydrology.Net](https://github.com/Harlinn/Harlinn.Windows/tree/master/DotNet/Libraries/Harlinn.Hydrology.Net) 
  which will be, when it grows up, a .NET library for hydrology applications. This is also an LLM code generation experiment
  where I am using AI tools to help me write scientific code for a domain with little existing code compared to common 
  line of business applications.
